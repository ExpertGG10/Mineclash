# Mineclash Datapack (Base)

## Estrutura
- `pack.mcmeta`
- `data/minecraft/tags/functions/load.json`
- `data/minecraft/tags/functions/tick.json`
- `data/mineclash/functions/load.mcfunction`
- `data/mineclash/functions/tick.mcfunction`

## Como instalar
1. Compacte **o conteúdo** da pasta `mineclash_datapack` em `.zip` (não compacte a pasta pai).
2. Coloque o `.zip` em `.../.minecraft/saves/<seu_mundo>/datapacks/`.
3. Entre no mundo e rode:
   - `/reload`
   - `/datapack list`

## Teste
- Ao recarregar, deve aparecer a mensagem:
  - `[Mineclash] Datapack carregado com sucesso!`
